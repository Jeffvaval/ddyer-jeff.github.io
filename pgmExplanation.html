A Simple Addition Program

By Jeff Vaval

What This Program Does

This C++ program adds two numbers using I/O. A variable named sum stores the result.

Code Example


int a, b, sum;
cin >> a;
cin >> b;
sum = a + b;
cout << sum;

If the input is 3 and 4, the output is 7.
