A Simple Cost Calculator Program

This page explains a small C++ program that calculates a total cost. The program uses I/O to receive numbers and produce a result.

A variable is a named storage location in a program. In this example, the variable total stores the final answer.

Code Fragment


double hotel, flight, meals, total;

cout << "Enter hotel cost:";
cin >> hotel;

cout << "Enter flight cost:";
cin >> flight;

cout << "Enter meal cost:";
cin >> meals;

total = hotel + flight + meals;
cout << total;

Input and Output Example

If the user enters 400, 300, and 200, the output will be 900. Notice how each variable contributes to the result.
